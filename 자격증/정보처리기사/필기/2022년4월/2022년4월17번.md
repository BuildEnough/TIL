## 문제
UI의 종류로 멀티 터치(Multi-touch), 동작 인식(Gesture Recognition) 등 사용자의 자연스러운 움직임을 인식하여 서로 주고받는 정보를 제공하는 사용자 인터페이스를 의미하는 것은?
1. GUI(Graphical User Interface)
2. OUI(Organic User Interface)
3. NUI(Natural User Interface)(O)
4. CLI(Command Line Interface)

## 풀이
# 011 사용자 인터페이스
사용자 인터페이스는 상호작용의 수단 및 방식에 따라 다음과 같이 구분된다
- CLI(Command Line Interface) : 명령이나 출력이 텍스트 형태로 이뤄지는 인터페이스
- GUI(Graphical User Interface) : 아이콘이나 메뉴를 마우스로 선택하여 작업을 수행하는 그래픽 환경의 인터페이스
- NUI(Natural User Interface) : 사용자의 말이나 행동으로 기기를 조작하는 인터페이스
- VUI(Voice User Interface) : 사람의 음성으로 기기 조작하는 인터페이스
- OUI(Organic User Interface) : 모든 사물과 사용자 간의 상호작용을 위한 인터페이스
