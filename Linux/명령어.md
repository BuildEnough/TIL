## ls 명령어 옵션
### 파일 속성과 상제 정보 출력
```
ls -l
```

### 숨겨진 파일 출력
```
ls -a
```

### 파일 종류 의미하는 기호를 추가하여 출력
```
ls -F
```

### 옵션 두 개 이상 지정
```
ls -al
```
```
ls -aF
ls -a -F
```

### 명령어의 인자도 지정하고 옵션도 지정
```
ls -a -F /
```
- `/`: 인자이고, 루트 디렉터리

### 문자 간격 맞춰 출력
```
ls -w30
ls -w 30
```

## 디렉터리, 파일
### 디렉터리 생성(work 디렉터리)
```
mkdir work
```

### 중접된 디렉터리 한 번에 생성
```
mkdir -p report/2024/07
```

### 파일 생성
```
touch newfile
touch newfile1 newfile2 newfile3
```

### 파일 삭제
```
rm newfile
rm newfile1 newfile2 newfile3
```

### 디렉터리 삭제
```
rm -r report
rmdir report
```

## 파일 내용 출력
### 파일 내용 출력
```
cat newfile
```

### 여러 파일 출력
```
cat newfile1 newfile2
```

### 행 번호 출력
```
cat -n newfile
```

## less 명령어(스크롤 표시)
### 파일을 스크롤하며 볼 수 있게 출력
```
less newfile.md
```

### 한 화면 아래로 스크롤
`Space`, `f`

### 한 화면 위로 스크롤
`b`

### 한 행 아래로 스크롤
`j`

### 한 행 위로 스크롤
`k`

### less 명령어 종료
`q`

### 문자열 검색
`/`  
- #### 다음 검색 결과 이동
    `n`
- #### 다음 검색 결과 이동
    `N`

## 복사
### 파일을 다른 이름으로 복사(file1을 복사)
```
cp file1 file2
```

### 파일을 특정 디렉터리 안에 복사(dir1안에 file1 복사)
```
cp file1 dir1
```

### 여러 파일을 디렉터리 안에 복사(dir1안에 file1, file2, file3을 복사)
```
cp file1 file2 file3 dir1
```

### 여러 파일을 디렉터리 안에 복사(dir1안에 txt 확장자인 것들을 복사)
```
cp *.txt dir1
```

### 파일 덮어쓰기 전 확인
```
cp -i file1 file2
```
- y/n 중 선택

### 디렉터리 복사
```
cp -r dir1 dir2
```
